
    
##  ASD-APP (Natural Language Processing to Motivate Expressive Language Development) – Capstone Project | Dec 2024
•	Developed an Android application to support children with Autism Spectrum Disorder (ASD) in enhancing expressive language skills.
•	Integrated Google Speech-to-Text API for real-time speech recognition and interactive feedback using natural language processing (NLP).
•	Implemented speech-to-image conversion with dynamic image management and customizable color options to reinforce language associations.
•	Implemented speech-to-image mapping using hash-mapping algorithm to associate objects with colors, sounds, and motion.
•	Developed a customizable UI, interactive animations, and optimized layout for tablet devices.
•	Enabled session tracking, user profiles, and image management (upload, rename, delete).

## Addressing Food Insecurity - Data Analysis | May 2024 
•	Analyzed food production, supply, and consumption trends across 45 African countries using large-scale datasets.
•	Cleaned and pre-processed data with Pandas, NumPy, and Geopy, ensuring naming consistency and data integrity.
•	Performed statistical analysis to uncover trends, including the effects of the 2007 global food crisis.
•	Built K-Means clustering models to group countries by food consumption patterns and highlight regional disparities.
•	Created interactive visualizations with Matplotlib and Plotly, including line charts, scatter plots, and choropleth maps.
•	Automated data processing pipelines with Python scripting, improving workflow speed and efficiency.
